# Wallet REST API

Разработка асинхронного API для управления кошельками пользователей.

## Стек технологий
- **Python 3.13**
- **FastAPI** — асинхронный фреймворк.
- **PostgreSQL** — основная база данных.
- **SQLAlchemy 2.0** + **asyncpg** — асинхронная работа с БД.
- **Alembic** — управление миграциями.
- **uv** — современный и быстрый пакетный менеджер.
- **Docker & Docker Compose** — контейнеризация.
- **Pytest** — тестирование.
- **Ruff** — линтинг и форматирование (соответствие PEP8).

## Ключевые особенности
1. **Конкурентность**: Реализована защита от Race Condition при параллельных запросах к одному кошельку с помощью **Pessimistic Locking** (`SELECT FOR UPDATE`). Это гарантирует целостность баланса при высокой нагрузке.
2. **Точность вычислений**: Для работы с деньгами используется тип `Numeric(12, 2)` в БД и `Decimal` в Python, что исключает ошибки округления float.
3. **Автоматизация**: Миграции применяются автоматически при запуске контейнера.

## Как запустить проект

1. Убедитесь, что у вас установлен Docker и Docker Compose.
2. Клонируйте репозиторий.
3. Запустите проект одной командой:
   ```
   docker-compose up --build

## Дополнительно

    ```    
    docker exec -it wallet_app-web-1 uv run pytest


## Эндпоинты
1. POST /api/v1/wallets — Создать новый кошелек (добавлен для удобства тестирования).
2. GET /api/v1/wallets/{WALLET_UUID} — Получить баланс кошелька.
3. POST /api/v1/wallets/{WALLET_UUID}/operation — Выполнить операцию (DEPOSIT/WITHDRAW).
